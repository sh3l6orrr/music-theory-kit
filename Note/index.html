Enumeration Note 

public enum Note

A custom note.

Create a Note either by using its name:

Note("D#")  // Note.D_sharp

or by directly using enum cases:

Note.B_flat

Flat and sharp versions are equivalent for the same note.

Note("C#") == Note("Db")  // true
